### :computer: &nbsp; Tecnologia utilizada

### :warning: &nbsp; Pré-requisito

Antes de executar as resoluções dos desafios, é necessário instalar o [JDK 11](https://jdk.java.net/java-se-ri/11) em sua máquina.
